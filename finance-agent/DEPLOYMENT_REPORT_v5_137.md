### 改進② 嚴格黑名單重入

**規則**:
```
被止損過的股票 → 冷卻30天 OR 質量分≥75
低勝率信號源(60日<40%) → 黑名單30天
```

**預期**: 虛假信號 35-40% → 27-32% (-8-13%)

### 改進③ 多週期數據緩存 (框架)

**架構**:
```
周線/月線數據 → 每天自動更新一次 → 避免實時重複
超時處理 → 使用上一有效狀態 → 不返回None
數據源降級 → 東方財富 → 新浪 → 本地緩存
```

**預期**: 數據採集成功率 ↑3-5%
