## 🎯 优化内容详述

### 优化 1：混合池赛道权重强化 (MIXED_POOL_SECTOR_WEIGHTS_V71)

**问题诊断：**
- 混合池 MACD+RSI 策略表现不佳：5.06% 收益率、0.86 Sharpe 比率
- 对标科技赛道 MACD+RSI：17.1% 收益率、2.35 Sharpe 比率（**差异 71%**）
- 根因：混合池选股缺乏赛道针对性，高效科技策略被低效消费策略拖累

**解决方案：**

在 `config.py` 新增赛道权重配置，在混合池选股时按回测数据加权不同赛道：

```python
MIXED_POOL_SECTOR_WEIGHTS_V71 = {
    '科技成长': 1.8,    # +80% 权重 (TOP1 策略)
    '新能源': 1.5,      # +50% 权重 (TOP2 策略)
    '消费白马': 0.5,    # -50% 权重 (低效赛道)
    '主板': 0.8,        # -20% 权重
    '其他': 0.7,        # -30% 权重
}
APPLY_MIXED_POOL_SECTOR_WEIGHTS = True
```

**预期收益：**
- 混合池收益率：5.06% → 8-10%（差异 -52%）
- Sharpe 比率：0.86 → 1.2+（改善 +40%）
- 集中投资于高效赛道，避免低效赛道污染
