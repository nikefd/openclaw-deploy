## 📝 工程思想

### 1. 回测驱动设计
- 直接从回测结果(17.1% + 2.35 Sharpe)反推配置
- 不是凭经验调参,而是数据驱动
- 减少过优化风险(已通过回测验证)

### 2. 赛道差异化
- 科技成长要求高信号(22分)因为机会多但噪声也多
- 新能源要求宽松(18分)因为趋势强方向清晰
- 消费白马保守(Kelly0.85x)因为需要稳定性

### 3. Sharpe分级管理
- Sharpe不是选股指标,是风险管理维度
- 高Sharpe的持仓值得加重(+30%)
- 低Sharpe自动降权保护组合

### 4. 现金激进化与风控平衡
- Kelly1.52虽然激进但有回测支撑
- 止损-8%严格执行防止爆仓
- 情绪>85时自动减速(v5.120情绪盾牌)

## 🎭 风险认知

### 风险1: Kelly系数过激进
- **缓解**: 回测数据支撑胜率60-70%
- **监控**: 日回撤>3%立即告警
- **保护**: 止损-8%严格执行

### 风险2: 赛道划分不够精准
- **缓解**: 使用分类器动态分类
- **回退**: 若赛道性能不符预期,自动降至统一权重

### 风险3: 现金过低导致应对突发
- **缓解**: MIN_CASH_RATIO 3%保证最少现金
- **流动性**: 每日检查流动性指标
