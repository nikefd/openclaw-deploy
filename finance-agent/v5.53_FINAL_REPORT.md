## 📝 关键设计理念

### 质的飞跃

**从 v5.52 → v5.53: "盲目追高" → "精准入场"**

```
原问题:
  · 只看"这只股好吗?" → 无法看"现在买好吗?"
  · 固定参数不适应市场 → 现金95%还严格过滤
  · 单一信号堆砌 → 信噪比低,误信号多
  · 参数脱离回测 → 实盘表现与回测反向

新方案:
  · 4维入场质量评分 → 看"好时机"
  · 动态松绑 → 根据市场状态自适应
  · 多维确认 → 趋势+位置+量价+动量
  · 直接融合TOP回测参数 → 实盘聚焦
```

### 数据驱动

```
参数来源:
  ✓ MACD+RSI 1.5x ← 回测TOP1 (17.1% Sharpe 2.35)
  ✓ 科技权重 0.30 ← 回测数据 (+50%)
  ✓ 入场质量4维 ← 历史胜率统计
  ✓ 过滤松绑 ← 现金比+连亏实时检测

自适应:
  ✓ 现金>95% → 自动激活宽松模式
  ✓ 连亏≥7次 → 自动微仓试单
  ✓ 市场环境 → 动态调权(regime)
```

### 风险管理

```
多层防护:
  1. 入场质量 ≥65分 (第一道关)
  2. 四维评分确保多角度验证
  3. 支撑位作为止损底线
  4. 连亏期固定微仓保留能力

渐进暴露:
  · 低质量入场 → 0.8x权重(排除)
  · 中等质量 → 1.0x权重
  · 高质量入场 → 1.3x权重(优先)
```
