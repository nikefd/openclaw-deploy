### 改进③ — 现金闲置激进模式 (position_manager.py)

**修改函数**: `kelly_position_size(sector, confidence)`

```python
# v5.50 新增逻辑
1. 获取当前现金比例 cash_ratio
2. 检查极度熊市标志:
   - 过去10日止损次数 > 5次 = 熊市信号
3. 激进条件判定:
   IF cash_ratio > 90% AND NOT is_extreme_bear:
      THEN:
      - params['max_kelly'] += 0.025  # Kelly上限+2.5%
      - adjusted *= 1.06              # 基数×6% + 上限调整
4. 返回调整后的仓位比例
```

**代码位置**: position_manager.py `kelly_position_size()` 第239行

**参数调整**:
- 现金阈值: 90% (保守触发)

- Kelly加幅: +2.5% (上限变更，如 0.25→0.35 / 0.15→0.175)
- 基数倍增: ×1.06 (+6%)

**预期效果**:
- ✅ 现金过剩时自动消耗，提高资金利用率
- ✅ 避免在熊市中激进加仓，风险控制

## ⚠️ 注意事项

### 1. RSI超卖信号的误触风险
- **问题**: 极端超卖不一定反弹，可能继续下行
- **对策**: 权重仅0.8x (保守) + 改进止损投范至8% (宽于常规6%)

- **问题**: 实时跌停股数据可能延迟
- **对策**: 用上证指数日跌幅作为备选信号 (更及时)

- **问题**: 10日内止损>5次 是否足够判定熊市?
- **对策**: 后续可升级为多因子熊市判定(MACD/ATR/成交量)
