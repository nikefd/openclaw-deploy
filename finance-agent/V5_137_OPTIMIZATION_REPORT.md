【优化1】止损风控参数优化
------------------------

当前: 固定点数止损 (华映4.10, 东方9.00)
问题: 
  • 固定点数不适应波动性变化
  • 小盘股波动性更大,容易被扫停损
  
优化:
  FROM: 固定止损 4.10 (华映)
  TO:   ATR动态止损 = 当日收盘 - 1.5*ATR14
  
实现:
  • 5分钟K线ATR14: 当前估计0.15-0.20
  • 动态止损: 4.26 - 1.5*0.18 = 4.03
  • 效果: 更灵活的风控,避免虚假止损

代码变更:
  config.STOP_LOSS_MODE = "DYNAMIC_ATR"  # 从 FIXED 改为 DYNAMIC_ATR
  config.STOP_LOSS_ATR_MULTIPLIER = 1.5
  config.STOP_LOSS_CHECK_INTERVAL = 5    # 5分钟检查一次

【优化2】现金配置动态模型
------------------------

当前: 固定现金目标 (97% 死水)
问题: 
  • 现金过多无法创造收益
  • 市场情绪72分时,应该更激进
  
优化:
  情绪分数 → 目标现金比例:
  
  情绪 > 80 (极度贪婪)  → 目标现金 15-25% (保持警惕)
  情绪 70-80 (贪婪)     → 目标现金 30-40% (适度保守)  
  情绪 50-70 (中立)     → 目标现金 45-55% (平衡配置)
  情绪 30-50 (恐惧)     → 目标现金 60-70% (主动加仓)
  情绪 < 30  (极度恐惧) → 目标现金 75%+  (大举加仓)

当前应用:
  情绪=72.5 → 目标现金 35-45%
  调整: 当前97% → 目标40%
  空闲现金: ¥972,193 × (0.97 - 0.40) = ¥552,996 可加仓
  
加仓安排:
  • Phase 1 (今日) : 加仓 +10% (¥100k) 选品种优质新龙头
  • Phase 2 (3天内): 加仓 +15% (¥150k) 技术确认后追加
  • Phase 3 (1周) : 加仓 +15% (¥165k) 达成40%现金目标

代码变更:
  config.EMOTION_BASED_CASH = True
  config.CASH_TARGET_MAP = {
    "panic": 0.75,     # 情绪<30
    "fear": 0.65,      # 30-50
    "neutral": 0.50,   # 50-70
    "greed": 0.35,     # 70-80
    "extreme": 0.20    # >80
  }


【优化3】选股模型加权调整
------------------------

当前: 基础选股 (一致性打分)
问题:
  • 华映失败: 资金面判断偏差
  • 龙虎榜数据缺失导致主力追踪困难
  
优化:
  技术面权重: 40% (MACD/布林带)
  资金面权重: 35% (当龙虎榜可用时) 或 25% (数据缺失时)
  情绪面权重: 15%
  基本面权重: 10% (同行业对比)
  
当前应用:
  龙虎榜缺失 → 资金面降权至25%
  强化成交量确认 (替代龙虎榜)
  
规则:
  • 突破必须伴随成交量增加(>150%日均)
  • 新进标的前3日严格止损(回撤>5%)
  • 技术指标需3个以上同步确认


【优化4】止盈策略改进
------------------------

当前: 达到目标价止盈 (东方10元)
问题: 
  • 被动止盈,缺乏主动性
  • 没有部分止盈机制,容易全部亏损
  
优化:
  分批止盈策略:
  
  东方证券 (600股 @ ¥9.23):
    • 第1批 (¥9.50): 卖100股 锁定小利
    • 第2批 (¥10.00): 卖200股 锁定中利
    • 第3批 (收益>10%): 卖150股 锁定大利
    • 保留: 150股 参与长期上涨

代码变更:
  config.PROFIT_TAKING_MODE = "PARTIAL"  # 从 SINGLE 改为 PARTIAL
  config.PROFIT_TARGETS = {
    "600958": [
      {"price": 9.50, "qty_pct": 0.17},  # 卖100/600
      {"price": 10.00, "qty_pct": 0.33}, # 卖200/600
      {"price": 10.50, "qty_pct": 0.25}  # 卖150/600
    ]
  }
