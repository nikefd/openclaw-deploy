## 🛡️ 第三阶段: 动态止损优化 ✅

### 方法对比

| 维度 | 固定止损(-8%) | ATR动态止损 | 优势 |
|-----|-------------|-----------|------|
| 机制 | 固定百分比 | 根据波动率 | ✅ATR更适应市场 |
| 保护 | 生硬 | 灵活 | ✅ATR保护更周全 |
| 利润 | 可能过早 | 允许合理波动 | ✅ATR保留更多利润 |
| 低波市场 | 容易被止 | 自动宽松 | ✅ATR防假跌 |
| 高波市场 | 保护不足 | 自动紧缩 | ✅ATR紧跟风险 |

### ATR配置

```python
DYNAMIC_STOP_LOSS_ENABLED = True
DYNAMIC_STOP_LOSS_METHOD = 'atr_adaptive'
ATR_PERIOD = 14           # 14天计算周期
ATR_MULTIPLIER = 2.5      # 止损线 = entry_price - 2.5*ATR
DYNAMIC_STOP_LOSS_MAX = 0.15  # 最大止损-15%(安全网)

计算公式:
  ATR(14) = 14日真实波幅平均
  止损价 = 入场价 - 2.5×ATR
  止损% = max((止损价-入场价)/入场价, -15%)
```

### 效果预期

- 低波市场(ATR<0.5%): 止损~3-4% (vs 固定8%)
- 中波市场(ATR~1%): 止损~5-7% (vs 固定8%)
- 高波市场(ATR>2%): 止损~8-12% (vs 固定8%)
- **整体**: 保留利润+保护回撤 = 更优的Sharpe比

## ⚠️ 风险分析

### 🔴 高风险项

1. **低入选门槛(15分) → 质量股数量激增**
   - 风险: 200+候选股,质量控制难度↑
   - 对策: 强化多维评分把关,加强单只头寸监控

2. **激进Kelly(1.60) → 放大收益&亏损**
   - 风险: 理论假设胜率60%,若实际50% → 负期望
   - 对策: 持续验证实盘胜率,月度K线评估

3. **高持仓数(12-15只) → 相关性风险**
   - 风险: 同赛道集中风险
   - 对策: 启用相关性监控,限制同赛道仓位

4. **动态止损(ATR) → 参数敏感性**
   - 风险: ATR倍数(2.5x)不当可导致过度止损或保护不足
   - 对策: 月度回测验证ATR参数,A<1.5时调整

### 🟡 中风险项

5. **情感指数可靠性**
   - 风险: 市场情感指数可能滞后或不准确
   - 对策: 配合技术面二次确认,不盲目跟随

6. **流动性压力**
   - 风险: 15只持仓×4.8%仓位 = 72%配置,现金仅28%
   - 对策: 设置日内现金预留,应对极端事件
