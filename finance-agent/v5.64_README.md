## 🎯 四大优化方向

### 1️⃣ 止损/止盈策略精细化

**函数**: `dynamic_stop_loss_by_sector()`

**机制**:
- ATR自适应 × 赛道系数 (科技1.5x, 消费0.8x)
- 流动性调整 (-1%)
- 科技赛道止盈提升 (20% → 22%)
- 熊市容错 (+1%)

**预期改进**: 最大回撤 4.08% → 3.2% (-20%)

---

### 2️⃣ 入场点优化 (Score+Timing)

**函数**: `best_entry_timing_check()`

**评分规则** (总分100):
- RSI < 30 (超卖反弹) → +15分
- MACD 金叉 → +10分
- 成交量 > 5日均×1.2 → +8分
- 高位 (>MA20×120%) → -5分

**评级**: OPTIMAL (≥20) | GOOD (10-20) | NORMAL (0-10) | AVOID (<0)

**预期改进**: 入场成功率 60% → 65% (+5%)

---

### 3️⃣ 风控增强 (多维检测)

**函数**:
- `position_correlation_check()` — 防相关性风险
- `leverage_market_detection()` — 识别融资杠杆
- `position_size_limit_check()` — 头寸数/占比限制

**规则**:
- 同赛道持仓: 最多3只 (科技最多2只)
- 融资余额>5000亿: 激进度-10%
- 持仓数: 最多20只
- 单只头寸: ≤10%

**预期改进**: 风险调整收益+30%, 回撤再降5%

---

### 4️⃣ 赛道权重微调 (动态调整)

**函数**: `sector_weight_by_winrate()`

**权重矩阵**:
```
科技成长: base=1.0  (Sharpe 2.35最高)
新能源:   base=0.80 (Sharpe 1.78, -20%)
消费:     base=0.70 (Sharpe 0.85, -30%)

× 实时胜率倍数:
  胜率≥60% → ×1.2
  胜率<40% → ×0.5 (反向惩罚)
```

**预期改进**: Sharpe维持2.35, 收益更均衡
