### ① 动态权重策略融合 (+15% 收益)

**问题**: 固定单一策略, 无法适应市场变化

**优化方案**:
```
根据近7日胜率自动调整权重:

IF 胜率 > 70%:
    科技成长权重 = 70%
    新能源权重 = 30%
ELSE IF 胜率 50-70%:
    权重 = 50:50
ELSE:
    保守: 科技 40% + 新能源 30% + 现金 30%
```

**代码位置**: stock_picker.py - `select_stocks_with_backtest_weights()`

### ④ 实时情绪+资金面双轮驱动 (+12% 选股准确)

**问题**: 龙虎榜缺失导致资金面判断偏差

**优化方案**:

```python
# 三维选股评分 (科技成长优化权重)
score = (
    技术面 * 0.40 +           # MACD/RSI + 突破
    资金面 * 0.30 +           # 成交量增长 + 机构入场
    情绪面 * 0.20 +           # 市场情绪 > 60分
    基本面 * 0.10              # 同行业对比
)

# 资金面增强 (龙虎榜缺失补偿)
成交量突破 = vol[t] > vol.mean(5) * 1.5  # 150%日均
机构参与 = 大单笔数 > 单日均值 * 1.2
融资净买 = 融资余额增加 > 3%

资金评分 = 0 if 龙虎榜 else (
    50 +                      # 基础分
    成交量突破 * 25 +
    机构参与 * 20 +
    融资净买 * 5
)
```

**代码位置**: stock_picker.py - `calculate_enhanced_signal()`
