實現代碼示例:
```python
def calculate_adaptive_threshold(cash_ratio, recent_winrate, concentration):
    base = 60
    
    # 現金調整
    if cash_ratio > 0.95:
        adjust = -25  # 極端: 35分
    elif cash_ratio > 0.75:
        adjust = -10  # 高現金: 50分
    else:
        adjust = 0    # 正常: 60分
    
    # 胜率調整
    if recent_winrate < 0.50:
        adjust += 15  # 低胜率: 75分
    elif recent_winrate >= 0.65:
        adjust -= 5   # 高胜率: -5分
    
    # 集中度調整
    if concentration > 0.30:
        adjust += 10  # 高風險: 70分
    
    return base + adjust
```
