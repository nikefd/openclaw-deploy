**實現機制**:
```python
def calculate_adaptive_threshold(cash_ratio, recent_winrate, concentration):

    # 現金調整

        adjust = -25  # 極端: 35分

        adjust = -10  # 高現金: 50分

        adjust += 15  # 低胜率: 75分
    # 集中度調整
    if concentration > 0.30:
        adjust += 10  # 高風險: 70分
    return base + adjust
```
