### 核心函數

```python
def get_dynamic_cash_target(sentiment_score: float) -> float:
    if sentiment_score < 25:
        return 0.03  # 極度恐懼
    elif sentiment_score < 40:
        return 0.05  # 恐懼
    elif sentiment_score < 60:
        return 0.15  # 中立
    elif sentiment_score < 75:
        return 0.25  # 貪婪
    else:
        return 0.40  # 極度貪婪
```

### 核心函數

```python
def get_adaptive_atr_multiplier(market_volatility: float = 1.0) -> float:
    if market_volatility < 0.5:
        return 1.8  # 低波:緊止損
    elif market_volatility < 1.5:
        return 2.5  # 中波:標準
    else:
        return 3.2  # 高波:寬止損
```

## 🛡️ 改進③: 極度貪婪防守 — 情感+技術背離雙重確認

### 防守觸發邏輯

```
觸發條件                             Kelly調整      持倉上限      防守強度
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
情感>92 + RSI>75 (雙重極端)         -30%          70%          🔴 高度防守
情感>92 單獨 (單一極端)              -15%          85%          🟠 中等防守
其他情況 (正常)                     無調整        100%          🟢 正常模式
```

### 核心函數

```python
def check_extreme_greed_defense(sentiment_score: float, rsi_value: float = 70) -> dict:
    if sentiment_score > 92 and rsi_value > 75:
        return {'kelly_reduction': 0.30, 'position_cap': 0.70}  # 極端
    elif sentiment_score > 92:
        return {'kelly_reduction': 0.15, 'position_cap': 0.85}  # 中等
    else:
        return {'kelly_reduction': 0.0, 'position_cap': 1.0}    # 正常
```

### Kelly系數調整示例

```
基礎Kelly = 1.60 (v5.124配置)

情況1: 情感95/100 + RSI78 (雙重極端)
  ├─ Kelly調整: 1.60 × (1-0.30) = 1.12 (-30%)
  ├─ 頭寸上限: MAX_POSITIONS × 0.70 = 10.5只 (vs 15只)
  └─ 效果: 激進→保守,降低極端風險

情況2: 情感95/100 單獨 (單一極端)
  ├─ Kelly調整: 1.60 × (1-0.15) = 1.36 (-15%)
  ├─ 頭寸上限: MAX_POSITIONS × 0.85 = 12.75只 (vs 15只)
  └─ 效果: 激進→中等,適度防守

情況3: 情感60-90 (正常)
  ├─ Kelly調整: 1.60 (無變)
  ├─ 頭寸上限: 15只 (無限)
  └─ 效果: 正常運作
```
