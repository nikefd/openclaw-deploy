#### 模块④: Kelly准则持仓优化 (V5_95_PositionOptimization)

基于回测数据精确配置单个持仓:

```
Kelly公式: f* = (bp - q) / b
  - b: 赔率 = 平均赢幅 / 平均亏幅
  - p: 胜率
  - q: 亏率 (1-p)

TOP1科技策略参考:
  - 胜率: 60% | 平均赢: 1.5% | 平均亏: 2.0%
  - 赔率 b = 1.5% / 2.0% = 0.75
  - Kelly% = (0.75 × 0.60 - 0.40) / 0.75 = 20%
  - Sharpe倍数(2.35): 1.5x → 20% × 1.5x = 30%
  - 安全约束: min(30%, 5% 单只限制, 8% 总上限) = 5%

建议持仓配置 (8只分散):
  - 前2只(科技): 5% 每只 = 10%
  - 中间3只(新能源): 4% 每只 = 12%
  - 后3只(医药/金融): 3% 每只 = 9%
  - 总部署: 31% | 现金预留: 69%
```

**效果**: 避免过度杠杆 | 科学仓位配置 | 风险-收益平衡
