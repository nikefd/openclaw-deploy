## 集成点1: score_and_rank() 风险过滤

在函数头部添加:
```python
def score_and_rank(candidates: list, regime: str = "", **kwargs) -> list:
    # ... 现有代码 ...
    
    # v5.56: 应用Sharpe风险权重
    for cand in candidates:
        strategy = cand.get('strategy', 'MULTI_FACTOR')
        risk_weight = get_strategy_risk_weight(strategy)
        
        if risk_weight == 0.0:
            cand['score'] = 0  # 黑名单
        else:
            cand['score'] = int(cand.get('score', 0) * risk_weight)
            cand['_strategy_risk_weight'] = risk_weight
```

## 集成点2: 赛道级策略权重

在score_and_rank()末尾添加:
```python
    # v5.56: 赛道级策略权重应用
    from position_manager import get_sector_optimal_strategy
    for cand in candidates:
        sector = cand.get('sector', '科技成长')
        routing = get_sector_optimal_strategy(sector)
        
        strategy = cand.get('strategy', '')
        if strategy.startswith('MACD'):
            weight = routing.get('primary', ('MACD_RSI', 0.65))[1]
        elif 'MULTI' in strategy:
            weight = routing.get('secondary', ('MULTI_FACTOR', 0.20))[1]
        else:
            weight = routing.get('hedge', ('MA_CROSS', 0.15))[1]
        
        cand['score'] = int(cand['score'] * weight)
        cand['_sector_weight'] = weight
```
