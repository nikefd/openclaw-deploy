### 🎯 改进②: Sharpe强制激活 ✅ **3.5倍权重确保生效**

**文件**: `v5_140_DEEP_EVENING_OPTIMIZE.py` → SharpeForceActivator 类

**核心逻辑**:
- 根据策略Sharpe比率计算加成:
  - Sharpe > 2.0 (TOP策略): +30分
  - Sharpe 1.5-2.0: +20分
  - Sharpe 1.0-1.5: +12分
  - Sharpe 0.5-1.0: +6分

- **3.5倍乘数应用**: 所有Sharpe加成 × 3.5倍
- **双重激活**: 在stock_picker的score_and_rank()和ranking()中都应用

**预期效果**:
```
TOP1策略(科技 Sharpe2.35) 的权重: 
  基础加成30分 × 3.5倍 = 105分加权提升
  确保科技成长赛道优先入场
```


### 🎯 改进⑥: 融资异变强制激活 ✅ **+12分底部/+8分参与**

**文件**: `v5_140_DEEP_EVENING_OPTIMIZE.py` → MarginAnomalyForce 类

**信号识别**:
1. **底部确认信号** (+12分):
   - 融资余额环比下降 > 20%
   - AND 融资融券比 < 20%
   - 应用场景: 强制离场冲出支撑→底部

2. **参与上升信号** (+8分):
   - 融资余额环比上升 > 15%
   - 应用场景: 资金抄底迹象→参与度提升

**强制应用** (无skip):
- 所有候选都会检查融资数据
- 满足条件自动+12分或+8分
- 不允许skip或降权

**预期效果**:
```
融资异变候选数: 预期日均2-3只
加权提升: 12-8分 × Sharpe倍数(3.5x) = 42-28分的额外权重
作用: 捕捉资金异动事件,抢占底部机会
```

**配置参数**:
