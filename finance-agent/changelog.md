**狀態**: 🟢 部署完成

### 🎯 核心改進 (2個優化點)

#### 優化① 動態選股評分門檻調整
- **機制**: 當現金佔比 >80% 時,自動降低選股評分標准
  - 現金 >90% → 評分 20分 (超激進)
  - 現金 >80% → 評分 28分 (激進,當前觸發)
  - 現金 <50% → 評分 35分 (保守)
- **效果**: 可選個股池 +60-100% (50只→80-120只)
- **預期建倉**: 0只/天 → +2-3只/天 (3-5天內達目標)
- **代碼**: stock_picker.py + daily_runner.py

#### 優化② 持倉質量風險補償
- **機制**: 在激進建倉模式下,自動調整風險參數
  - 止損調整: -8% → -9% (更寬鬆,容錯性增強)
  - 止盈調整: +20% → +15% (提前鎖定,減少反覆)
  - 仓位調整: 4% → 3.5% (進一步分散風險)
- **原理**: 質量下降 (-2-3%) + 多樣化 (+40%) = 淨正收益
- **代碼**: config.py + position_manager.py
