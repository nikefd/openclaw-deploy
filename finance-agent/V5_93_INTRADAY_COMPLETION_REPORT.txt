【技術實現細節】

1. 現金激活邏輯:
   - cashRatio > 95% → 超激進 (3.5x)
   - cashRatio > 75% → 平衡   (2.0x)
   - cashRatio ≤ 75% → 保守   (1.0x)

2. 入場質量評分:
   - 讀取最近30筆 trades.entry_quality_score
   - 平均值作為當前質量分數

3. MACD信號計數:
   - 24h: 過去24小時內 reason 包含 'MACD' 的交易數
   - 7d:  過去7天內 reason 包含 'MACD' 的交易數
   - 週均: max(1, round(7d/7))

4. 熱力圖色彩:
   - pnl_pct ≥ 10%   → green
   - pnl_pct ≥ -5%   → yellow
   - pnl_pct ≥ -10%  → orange
   - pnl_pct < -10%  → red
