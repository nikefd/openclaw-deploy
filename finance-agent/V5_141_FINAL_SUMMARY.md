## 📋 核心算法更新

### 信号融合公式
```
最终评分 = 信号融合75% + AI补偿25%

其中:
  信号融合 = Σ(技术/资金/情绪/基本 × 动态权重)
  动态权重 = f(市场情绪, 波动率)
  AI补偿 = 基础20 + Σ(5维信号 × 权重)
```

### 状态机转移
```
IF 情绪得分变化 > 5点 THEN
  current_state = GET_STATE(new_sentiment)
  config = STATE_CONFIGS[current_state]
  APPLY_CONFIG(kelly, trailing_stop, max_positions, ...)
  LOG_TRANSITION(old→new, timestamp, sentiment)
ENDIF
```

## 🚨 已知限制

1. **AI补偿依赖质量**
   - 需要实时成交数据、融资余额、机构大单
   - 数据缺失时使用默认基础分

2. **状态机敏感度**
   - 情绪波动>5点才转移 (防止过度反应)
   - 预热期需要3-5个交易日

3. **权重调整平滑**
   - 波动率调整仅在极端情况(<15% 或 >50%)应用
   - 常规范围(15-50%)保持基础权重
