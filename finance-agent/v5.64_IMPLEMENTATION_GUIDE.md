## 🔍 故障排除

### Issue 1: 导入失败 "ModuleNotFoundError"

**症状**: `ImportError: No module named 'v5_64_...'`

**解决**:
```bash

# 重新导入
python3 -c "import sys; sys.path.insert(0, '/home/nikefd/finance-agent'); from v5_64_deep_optimize_functions import *"
```

### Issue 2: 数据库查询异常

**症状**: `sqlite3.OperationalError: no such column: sector`

**原因**: trades表可能没有sector列

**解决**:
- 使用`position_manager.py`中的`classify_sector()`函数动态计算
- 所有函数都有try-except降级

### Issue 3: 性能下降

**症状**: 选股或回测速度变慢

**原因**: 新增函数的数据库查询

**解决**:
- 增加缓存: 在daily_runner.py中缓存sector_weight_by_winrate()的结果
- 可选: 禁用 `sector_weight_dynamic`，只启用关键优化

```python
# 在 config.py 中
V5_64_OPTIMIZATIONS_ENABLED = {
    'dynamic_stop_loss': True,        # 必启用

    'leverage_detection': True,       # 可选
    'sector_weight_dynamic': False    # 如性能受影响可禁用
```
