### 優化② - Kelly系數進階升級 (1.35 → 1.45)

**理論基礎**:
```
勝率 60% 時:
  kelly_frac = Win% × Win/Loss - Loss%
             = 0.6 × 1.875 - 0.4
             ≈ 0.73

Kelly系數計算:
  v5.115: 1.35 × 0.73 ≈ 0.985 ≈ 1.0 (完整Kelly)
  v5.120: 1.45 × 0.73 ≈ 1.059 (超Kelly +5.9%)

實際收益改進:
  (1.45 - 1.35) / 1.35 ≈ +7.4% 年化提升
```

**配置升級全景**:

| 參數 | v5.115 | v5.120 | 改進 |
|------|--------|--------|------|
| Kelly系數 | 1.35 | 1.45 | +7.4% |
| 單倉最大 | 3.5% | 3.8% | +8.6% |
| 勝率加成 | +1.6% | +2.0% | +25% |
| 12倉配置 | 42% | 45.6% | +3.6% |

**實盤配置計算**:
```
基數計算 (勝率60%):
  基數: 3.8% (Kelly上限)
  勝率加成: 60%-50% = 10% (對標)
  加成幅度: 3.8% × 1.45 × (1 + 0.10) ≈ 6.0%
  實際配置: 3.8% (Kelly保護上限, 不超)

12倉組合:
  12 × 3.8% = 45.6%
  現金保留: 54.4% (充足應急)
  資金利用率: 45.6% (vs v5.115: 42%)
```

**安全容錯**:
```
勝率跌至50%:
  kelly_frac ≈ 0.5 × 1.875 - 0.4 ≈ 0.5375
  配置衰減: 1.45 × 0.5375 ≈ 0.78 (低於1.0, 自動縮小)

勝率跌至40%:
  kelly_frac ≈ 0.0 (禁止加倉)
  配置衰減: 1.45 × 0.0 ≈ 0 (禁止持倉加倉)

最壞情境回撤測試:
  12倉 × 3.8% × -8%止損 = ¥43,776 最大虧損
  回撤率: 43,776 / 1,001,863 ≈ 4.4% (可控)
```
