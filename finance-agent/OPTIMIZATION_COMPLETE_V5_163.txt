## 📋 功能詳解

### ①️⃣ 實時P&L彩色儀表板

**特性**:
- 🟢 **風險分級**: 4級顏色編碼 (低/中/高/臨界)
- 📊 **持倉卡片**: 成本、現價、P&L、持有天數、質量分數
- 📈 **回撤監控**: 峰值回撤 (peak_dd) 實時追蹤
- 🔢 **組合評分**: 0-100風險分數

**API結構**:
```json
{
  "pnl_dashboard": {
    "positions": [
      {
        "symbol": "AAPL",
        "pnl": 500.00,
        "pnl_pct": 5.00,
        "risk_level": "low",
        "entry_quality": 75
      }
    ],
    "summary": {
      "total_pnl": 5000.00,
      "utilization_pct": 65.0,
      "risk_score": 15
    }
  }
}
```

### ②️⃣ 盤中風險警告儀表板

**三級警告系統**:
1. 🚨 **臨界** (STOP_LOSS_ALERT)
   - 虧損 > 止損設置
   - 一鍵止損按鈕

2. ⚠️ **高危** (APPROACHING_STOPLOSS)
   - 虧損 ≥ 止損70%
   - 距離提示

3. 💡 **低優先** (TAKE_PROFIT_SIGNAL)
   - 盈利 > 5% AND 質量 > 70
   - 鎖定利潤建議

### ③️⃣ 交易信號實時推送

**推送內容** (5分鐘內最新20筆):
- 交易代碼 + 方向 (BUY/SELL)
- 信號類型 + 成交價
- 實時P&L + 百分比
- 狀態 (OPEN/CLOSED)

### ④️⃣ 回測系統增強

**交易頻率分析**:
- 日均交易筆數
- 日平均收益%
- 日勝率

**信號持久度報告**:
- 優秀(80+) / 良好(70-79) / 中等(60-69) / 較弱(<60)
- 各等級: 交易數、勝率、最佳/最差交易
