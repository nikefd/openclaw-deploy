## 🎯 三大改進詳解

### 改進①: RSI信號持續性驗證 ✨

**問題分析**:
- v5.88/v5.90 只對 MACD+RSI 組合做持續性驗證
- **單純RSI信號缺乏驗證**，易捕捉噪聲 (超買/超賣區振盪)
- 當前誤觸率: 15-20%

**解決方案**:

```python
verify_rsi_signal_persistence(code) 
    ↓ 檢查3項指標:
    ├─ RSI在極端區(>70/<30)停留天數 (需≥2天)
    ├─ RSI方向一致性 (需≥2個交易日同向)
    └─ 價格與RSI同步性 (價格漲RSI應升)
    
    ↓ 返回:
    {
        'is_persistent': bool,        # 是否持續
        'confidence': float,          # 信心度 (0-1)
        'rsi_days_extreme': int,      # 極端區天數
        'rsi_direction_consistent': bool,
        'price_rsi_sync': bool
    }
```

**應用邏輯**:
- RSI 不持續 (極端區<2天) → 折扣 **30%**
- RSI 低信度 (<0.65) → 折扣 **15%**
- RSI 持續且高信度 → 無折扣 ✓

**預期改善**:
- 信噪比: 85% → 90% (+5%)
- 日均虛擬倉: 12-18只 → 11-15只 (-清雜訊)
- 命中率: 維持 70-75%
