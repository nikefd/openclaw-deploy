## 📊 优化内容

### 1. 新增API端点: `/api/finance/performance-stats`

**功能**: 实时绩效统计聚合

```javascript
{
  win_rate: 60,              // 胜率 (%)
  max_drawdown: 4.08,        // 最大回撤 (%)
  sharpe_ratio: 2.35,        // Sharpe比率
  total_trades: 31,          // 总交易数
  positions_count: 2,        // 当前持仓数
  monthly_return: 2.45,      // 月度收益 (%)
  sectors: {                 // 赛道分布
    "科技": 1,
    "金融": 1
  }
}
```

**实现**:
- ✅ 从trade数据计算胜率
- ✅ 从daily_snapshots计算最大回撤 (30天)
- ✅ 计算Sharpe比率 (收益/风险调整)
- ✅ 按赛道分类头寸

### 2. 前端UI增强脚本: `ui-optimize-v5.65.js`

**新增组件**:

#### 2.1 绩效统计仪表板 (6大卡片)
```
┌─────────────┐  ┌──────────┐  ┌──────────┐
│ Sharpe     │  │  胜率    │  │  回撤    │
│ 2.35 🟢    │  │ 60% 🟡  │  │ 4.08% 🔴 │
└─────────────┘  └──────────┘  └──────────┘

┌─────────────┐  ┌──────────┐  ┌──────────┐
│ 月度收益   │  │ 交易数   │  │ 风险等级  │
│ +2.45% 🟢  │  │    31    │  │   中 🟡  │
└─────────────┘  └──────────┘  └──────────┘
```

- 动态颜色编码: 🟢绿(低风险) / 🟡黄(中等) / 🔴红(高风险)
- 目标值对标显示
- 2分钟自动刷新

#### 2.2 赛道权重分布面板
```
赛道权重分布
┌──────────┐ ┌──────────┐
│  科技    │ │  金融    │
│   1头寸  │ │  1头寸   │
└──────────┘ └──────────┘
```

#### 2.3 风险等级自适应评分
```
评分规则:
- MaxDD < 3% && Sharpe > 2   → 🟢 低
- MaxDD < 8% && Sharpe > 1   → 🟡 中  
- 其他情况                   → 🔴 高
```
