## 🎓 技術深度

### 多策略融合數學模型
```
綜合Sharpe = Σ(Sharpe_i × Weight_i)
           = 2.35 × 0.65 + 1.78 × 0.25 + 1.66 × 0.10
           = 2.139

目標: 通過權重組合達到穩定的2.14-2.35範圍
```

### Kelly系數二維調整
```
Kelly_Final = Kelly_Base 
            × Sentiment_Adjustment(情感)
            × Sector_Adjustment(行業)
            × Sharpe_Adjustment(品質)
            × Drawdown_Adjustment(風險)

例子: 極度恐懼 + 科技成長 + Sharpe高 + 回撤小
    = 1.60 × 1.25 × 1.15 × 1.10 × 1.15
    = 3.00 (激進配置)
```

### ATR止損精準公式
```
StopLoss = EntryPrice - ATR_Multiplier × ATR
         - Sharpe_Adjustment × ATR_Multiplier × ATR
         - Drawdown_Adjustment × ATR_Multiplier × ATR

例子: 科技成長 + Sharpe高 + 回撤小
    = 100 - 3.0×5 - 0.10×3.0×5 - 0.15×3.0×5
    = 100 - 15 - 1.5 - 2.25
    = 81.25 (寬容止損)
```
