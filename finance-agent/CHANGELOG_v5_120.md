### 優化② - Kelly系數進階升級 (1.35 → 1.45)

**改進邏輯**:
```
v5.115現況: Kelly系數1.35
           單倉上限3.5%
           12倉配置42%

理論分析: 胜率60% 下
         kelly_frac = 0.6×1.875 - 0.4 ≈ 0.73

Kelly系數調整:
  1.35 × 0.73 ≈ 0.985 ≈ 1.0 (完整Kelly)
  1.45 × 0.73 ≈ 1.059 (超Kelly +5.9%)

實際收益預期差: (1.45-1.35)/1.35 ≈ +7.4%
```

**配置升級**:
```python
# v5.115
KELLY_COEFFICIENT = 1.35
KELLY_MAX_POSITION = 0.035       # 3.5%
KELLY_WIN_RATE_BOOST = 0.08      # 每5%勝率 +1.6%

# v5.120 (當前)
KELLY_COEFFICIENT = 1.45         # +7.4%
KELLY_MAX_POSITION = 0.038       # 3.8% (+8.6%)
KELLY_WIN_RATE_BOOST = 0.10      # 每5%勝率 +2% (+25%)
```

**新仓位计算**:
```
單倉基數: 3.8% (vs v5.115: 3.5%)

勝率60%時:
  基數 3.8% × 1.45系數 × 1.06加成 (60%-50%) ≈ 5.9%
  實際配置: 3.8% (Kelly保護上限)

12倉組合配置:
  12 × 3.8% = 45.6% (vs v5.115: 42%)
  現金保留: 54.4% (vs v5.115: 58%)

資金利用率提升:
  45.6% - 42% = +3.6% 進一步激進化
```

**安全容錯**:
```
胜率跌至50%時:
  kelly_frac = 0.5×1.875 - 0.4 ≈ 0.5375
  1.45 × 0.5375 ≈ 0.78 (低於1.0, 自動保護!)
  實際配置衰減至 60% vs 基數

最壞情境:
  胜率40%: kelly_frac ≈ 0.0 (禁止加倉)
  實際配置: 3.8% × 0.3 ≈ 1.1% (極度保守)
```
