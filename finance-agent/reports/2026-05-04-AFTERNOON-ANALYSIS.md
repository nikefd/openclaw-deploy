## 💡 优化方案 (v5.83)

### 优化A：资金配置策略 (立即实施)

**目标仓位结构（总资产）**

```
防守仓位 35%   ← 东方证券 (已有) + 稳定性强的蓝筹
  → 1-2只大盘蓝筹 (A股、工行类)，目标+2-5%年化

进攻仓位 40%   ← 热点轮动板块
  → 半导体/军工/锂电中挑3-5只，目标+15-30%年化

机动仓位 15%   ← 低位补涨+高分红
  → 分散风险，防守反弹

现金仓位 10%   ← 应对突发机会或风险
  → 关键技术位突破或暴跌时快速出击
```

**对比当前：**
- 当前: 防守35% + 进攻0% + 机动0% + 现金65% ❌ 太保守
- 目标: 防守35% + 进攻40% + 机动15% + 现金10% ✅ 平衡

### 优化B：入场逻辑改进 (编码实施)

**新的三层筛选机制：**

```
第1层（粗选）: 动量+资金+技术三合一
  ✓ MACD金叉 或 RSI 30-70区间
  ✓ 大笔买入 ≥2笔（资金信号）
  ✓ 最近3日交易量 > 平均20日 × 1.2
  → 产出: ~80-100只候选

第2层（精选）: 板块热度+主力意图
  ✓ 所在板块主力净流入占比 ≥3%
  ✓ 个股超大单净流入 ≥5000万
  ✓ 最近20日涨幅 20-100% (避免极端高位)
  → 产出: ~20-30只推荐

第3层（分配）: 分批建仓，控制单只权重
  ✓ 每只初始配置 2-3% 基金
  ✓ 首批买入 (情绪>70): 立即买入
  ✓ 次批买入 (情绪60-70): 等回调3%后买入
  ✓ 底仓持有: 绝不频繁止损，至少持有5个交易日
  → 产出: 3-5只仓位，避免single-stock集中风险
```

### 优化C：止损与止盈规则 (v5.83编码)

**新的动态止损止盈**

| 情景 | 触发条件 | 操作 | 逻辑 |
|------|--------|------|------|
| 快速获利 | 进入后3日内涨 ≥8% | 止盈50% | 锁定利润，防止反转 |
| 正常持有 | 进入后3-10日 | 持有，目标+15% | 给予趋势充分时间 |
| 技术止损 | 跌破进价 -3% | 割肉25% | 快速认亏，保护本金 |
| 底部加仓 | 跌破进价 -5% 且情绪<50 | 加仓50% | 价值区域逆向操作 |
| 终极止损 | 跌破进价 -8% | 全部清空 | 规避系统性风险 |

**对比当前：** 之前是-5%就割肉，容易被洗出。新方案更有耐心。

```python

   → 当前仅1.3%，远低于目标

5. Sharpe比率 (Target: ≥1.5)
   = 日均收益 / 日波动 × √252
```
