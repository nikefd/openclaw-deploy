### 1️⃣ position_manager.py (+350行新增函数)

**新增4个关键函数:**

| 函数 | 功能 | 预期效果 |
|------|------|--------|
| `get_cash_utilization_rate()` | 计算现金占比(0-1.0) | 作为激进模式判断依据 |
| `check_position_adding_condition()` | 检查加仓条件 | 胜者加仓,风控 |
| `check_trailing_stop_loss()` | 追踪止损检查 | 保护盈利,防止回撤 |
| `get_extreme_cash_mode_boost()` | 获取超激进权重加成 | 现金>98%时2.2x权重 |

**关键逻辑:**

a) **现金利用率计算** (get_cash_utilization_rate)
```python
cash_util = cash / total_value  # 从daily_snapshots读取
返回值: 0.98代表98%现金占比
```

b) **加仓条件** (check_position_adding_condition)
```
✓ 持仓≥3天
✓ 浮盈>2%
✓ 技术面≥2个强信号
→ 加仓 = 原头寸 × Kelly系数 × 50%
```

c) **追踪止损** (check_trailing_stop_loss)
```
✓ 从峰值回撤>5% → 触发
✓ 8小时无新高 → 触发
→ 锁定95%峰值收益
```

d) **超激进权重** (get_extreme_cash_mode_boost)
```
现金>98% → MACD_RSI: 2.2x, MULTI_FACTOR: 1.4x, ...
现金90-98% → 正常权重1.8x-1.2x
```

## 🔧 技术细节

### 权重加成体系

```
基础分数 (score_and_rank原始值)
    ↓
+ Sharpe权重倍数 (1.5x或2.0x)
    ↓
+ 现金占比激进系数 (ultra_high: 2.2x MACD_RSI)
    ↓
+ 入场质量评分乘数 (0.7x-1.3x)
    ↓
= 最终加权分数
```

**示例计算:**
- 原始MACD_RSI信号: 50分
- Sharpe倍数(2.0x): 50 × 2.0 = 100分
- 现金占比加成(2.2x): 100 × 2.2 = 220分
- 入场质量(1.1x): 220 × 1.1 = 242分
- **最终分数: 242分** (vs原始50分, +384%)
