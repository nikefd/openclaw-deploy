### 优化1: 扩展策略池 - 新增3个高Sharpe策略
**目标**: 提高策略多样性, 覆盖更多赛道

#### 策略A: 动量+情绪组合 (MOMENTUM_SENTIMENT)
- **原理**: 利用市场情绪反转 + 价格动量
- **参数**:
  - Momentum周期: 20天
  - RSI情绪阈值: 30(超卖) / 70(超买)
  - 情绪权重: 0.3x (防止过度拟合)
- **预期Sharpe**: 1.9-2.1
- **适用赛道**: 白马消费(防御) + 金融(周期)

#### 策略B: 均线反转+波动率加权 (MA_REVERT_VOL)
- **原理**: 偏离均线时进场 + 低波动率时增仓
- **参数**:
  - MA周期: 120天(中长期)
  - 偏离度: ±2.5%
  - Sharpe权重: 1.2x
- **预期Sharpe**: 1.7-1.9
- **适用赛道**: 金融/地产(稳定现金流)

#### 策略C: 隐含波动率套利 (IV_ARBITRAGE)
- **原理**: 买入被低估的IV, 卖出高估的IV
- **参数**:
  - IV百分位: <30% 买 | >70% 卖
  - 头寸大小: 风险平价
- **预期Sharpe**: 2.0-2.3
- **适用赛道**: 高流动性蓝筹(000858/002594等)

### 优化4: 动态止损+风险调整头寸 (高级风险管理)
**目标**: 主动降低回撤, 保护资本

#### 功能A: 智能止损算法 (Smart Stop Loss)
```
对于每个持仓:
  1. 计算该股票的ATR (Average True Range)
  2. 设置止损位 = 入场价 - 2.5*ATR (动态, 不是固定%)
  3. 当跌破止损位时:
     - 若Sharpe > 1.5: 保留30%头寸(长期看好)
     - 若Sharpe < 1.0: 清仓(质量不够)
  4. 每周review一次
```

#### 功能B: 回撤触发保护 (Drawdown Protection)
```
全组合实时回撤监控:
  - 回撤 < 2%: 正常运作
  - 回撤 2-3%: ⚠️  暂停新建仓, 调整止损 -0.5%
  - 回撤 3-5%: 🔴 减仓20%, 止损 -1%
  - 回撤 > 5%: 🛑 紧急清仓30-50%
```

#### 功能C: 头寸大小优化 (Position Sizing by Kelly Criterion)
```
Kelly最优头寸大小 = (Sharpe² / (1 - win_rate)) * f
- f = 激进系数 (0.5-1.0)
- 根据市场情绪自动调整f: 
  - 情绪>85 (极度贪婪): f = 0.3x
  - 情绪70-85 (贪婪): f = 0.5x  
  - 情绪40-70 (正常): f = 0.8x
  - 情绪<40 (恐惧): f = 1.0x
```
