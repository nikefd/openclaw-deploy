**预期**: TOP赛道 +2-5%, 低效赛道有效隔离

### 改进2️⃣: 风险管理强化 (200行代码)
**问题**: v5.80用固定止损，浮盈无保护
**解决**: 加入资金曲线实时监控 + 2级自动减仓

```
回撤范围    触发条件         动作
──────────────────────────────────────
0-3%       正常             HOLD (继续建仓)
3-5%       轻度回撤         REDUCE 10% (保护浮盈)
5%+        重度回撤         ALERT + REDUCE 30% (保护本金)
```
**预期**: MaxDD 4.08% → 3.5% (-14%), 浮盈保护 +8%

### 改进3️⃣: 入场质量优化 (150行代码)
**问题**: v5.80全局25分入场，胜率57%
**解决**: 分层入场策略，根据赛道+Sharpe进行差异化

```
层级           分数范围  赛道限制        预期胜率
──────────────────────────────────────────────
Tier1_Fast     22-24    科技成长        55% (快速激进)
Tier2_Std      25-26    科技/新能源     57% (平衡)
Tier3_Cau      27-30    其他赛道        65% (谨慎确认)
Tier4_Confirmed 31+     所有赛道        80% (极低风险)
```
**预期**: 整体胜率 57% → 62-65% (+8-14%)

### 改进4️⃣: 赛道实时监控 (150行代码)
**问题**: v5.80每周评估赛道一次，响应滞后
**解决**: 日级别指标跟踪 + 异常自动检测 + 轮换建议

```
监控维度          告警阈值         触发动作
────────────────────────────────────────────
Sharpe (5d avg)   <1.0 (警告)      观察
                  <0.5 (紧急)      建议轮换

MaxDD (5d max)    >8%              立即告警

回撤加速度        连续2日-5%       风险预警
```
**预期**: 风险发现时间 周级 → 日级, 系统稳定性 +12%
