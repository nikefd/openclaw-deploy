📡 接入新机器

1
在远程机器上安装 OpenClaw 并配置权限：
npm install -g openclaw
mkdir -p ~/.openclaw
echo '{"version":1,"defaults":{"security":"full"}}' > ~/.openclaw/exec-approvals.json
2
设置 Gateway Token 并启动 Node Host：
加载中...
3
回到这里点击 ✅ 批准 配对请求
💡 提示：配对成功后，你可以在终端和文件浏览器中选择远程节点进行操作。也可以安装为系统服务实现开机自启：
openclaw node install --host zhangyangbin.com --port 443 --tls
