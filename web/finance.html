🎯 当前策略体系

金融Agent采用多策略融合+AI决策的选股体系，以下是各模块详情

📋 选股流程
① 6策略信号采集 → ② 综合打分排名 → ③ 技术面验证 → ④ 新闻+资金面叠加 → ⑤ AI最终决策 → ⑥ 动态仓位执行
📈 策略1: 动量选股

数据源: 同花顺

信号:

  • 量价齐升 — 连续N天量价同涨，趋势延续信号
  • 创新高 — 突破前期高点，突破确认信号

打分: 量价齐升天数×3 + 创新高+10

回测表现: 在科技成长股上效果最佳

💰 策略2: 资金追踪

数据源: 东方财富异动

信号:

  • 大笔买入 — 统计同一股票出现次数(≥3次)
  • 火箭发射 — 快速拉升信号(≥2次)

打分: 大笔买入次数×3 + 火箭次数×4

逻辑: 跟着主力资金走，多次大买=持续看好

🔥 策略3: 强势股池

数据源: 涨停板/连板数据

信号:

  • 涨停股 — 当日封板成功
  • 强势连板 — 连续多日涨停

打分: 基础8分

⚠️ 风险: 已涨停的会被过滤(买不进)，主要用于辅助判断市场方向

📋 策略4: 机构推荐

数据源: 东方财富研报中心

信号:

  • 买入评级 — 机构给出"买入"评级 (+12分)
  • 增持评级 — 机构给出"增持"评级 (+8分)

打分: 买入+12, 增持+8

逻辑: 专业研究员背书，基本面有支撑

📰 策略5: 新闻/舆情 NEW

数据源: 6源聚合

  • 东方财富资讯 — 综合财经新闻 (~50条)
  • 财联社电报 — 最快A股实时消息 (~30条)
  • 政策公告 — 央行/证监会/国务院 (~17条)
  • 股吧热度 — 散户情绪风向标 (TOP20)
  • 个股新闻 — akshare聚合 (~10条)

LLM解读: 非结构化新闻→结构化交易信号

打分: 利好+15, 利空-20, 板块利好+8, 板块利空-10

💰 策略6: 资金面 NEW

数据源: 4大机构级数据

  • 北向资金 — 外资净流入+个股增减持 (+12/-15分)
  • 龙虎榜 — 机构/游资真金白银 (+15/-12分)
  • 融资融券 — 杠杆资金动向 (+5分)
  • 宏观经济 — PMI/M2/社融/CPI

综合评分: 资金面0-100分，叠加到个股打分

💰 动态仓位管理

基础仓位 (按信心评分):

10分→15% 9分→13% 8分→12% 7分→10% 6分→8%

情绪调节:

  • 极度贪婪(>85) → 仓位×0.6 减仓
  • 乐观(>70) → 仓位×0.85
  • 恐慌(<30) → 仓位×1.3 逆向加仓
  • 谨慎(<45) → 仓位×1.1

限制: 单只≤15%、总持仓≤10只

🛡️ 风控规则

止损 (动态):

  • 默认: -8%
  • 市场恐慌时: -5% (收紧保护)
  • 市场乐观时: -10% (给空间)

止盈 (阶梯):

  • 盈利≥20% → 卖出一半 (锁定利润)
  • 盈利≥30% → 全部卖出

过滤:

  • 自动排除: ST股、停牌股、涨跌停(买不进)
  • T+1规则: 当日买入次日才能卖
  • 佣金万三 + 印花税千一(卖出)
🤖 AI最终决策

多策略打分排名后，Top 10候选送入LLM做最终判断：

  • 优先选: 多信号叠加(量价齐升+大笔买入+MACD金叉)的品种
  • 避开: RSI>80严重超买 / 已涨停 / 连续涨停3天+
  • 分散: 不全选同一板块
  • 结合: 新闻/研报/资金面综合研判

模型: Claude Opus 4.6 (通过OpenClaw Gateway)
